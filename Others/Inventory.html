Bagel's Inventory Documentation

Adding Items

Which Methods to Use

When adding an item to the inventory that comes from a predefined item (the scriptable object), it’s best to use Inventory.AddItem(ItemObject.GetItem());

If the item exists in-game, and it relies on a gameobject to exist, and said gameobject WILL be destroyed, you should use Inventory.AddClonedItem(itm);
This will generate a duplicate of the item, so that destroying the gameobject the item is attached to doesn’t destroy the reference as well.

Removing Items

To remove an item, simply use Inventory.RemoveItem(GUID, amount), where the GUID is the GUID of the item. If the item is stackable, the GUID will be the same as the item's name. Amount is not a required parameter, and it defaults to 1 if not provided.

Getting items

The best practice to use when getting an item is Inventory.GetItemByGUID(GUID). The alternative, and much slower approach, would be to use Inventory.GetItemByName(Name).

Item structure

The Item class, located in ItemObject.cs, is a serializable class with a few important attributes. The first attribute is the Name, which is simply the Name of the item. The second attribute is the Stackable boolean. This should be true if the item can be stacked. Then, there's the GUID, which is a random globaly unique identifier assigned to the item. This can be auto-generated with Item.RegenerateGUID(). If the item is stackable, the GUID will be the same as the item's name.
Finally, there's the Amount value. If the item is stackable, this amount will be used when adding the item to the inventory.

Additional Attributes

Additional attributes can be added to the item using the Properties attriubute. This is a list of custom properties for the item. Each property can contain a string value, as well as a floating point value. To access a property, use either Item.GetPropertyTxt(propertyName) to get a string value, or Item.GetPropertyValue(propertyName) to get a float value.


The example scene

Click on the Canvas object to view the inventory in action. Press "1" to add a non stackable item. Press "2" to add a stackable item. Press "3" to remove the stackable item. Press "4" to update the UI view.


Complete method documentation

MethodDescription
AddItem(Item itm)Adds an item to the inventory.
AddClonedItem(Item itm)Adds a clone of the given item to the inventory.
RemoveItem(string guid, int amt)Removes an item from the inventory.
GetItemByGUID(string GUID)Gets an item using the given GUID.
GetItemByName(string Name)Gets an item using the given Name. Will return the first item found with a matching name.
