Bagel Engine 3: Getting Started

Bagel Engine 3

To start, create a canvas and stuff. Be sure to give it the ID "Canvas"

<canvas id="Canvas" style="margin: 0 auto; background: darkgray; display: block;" width="300px" height="300px"></canvas>

Next, add the necessary scripts to your html file (order matters)

<script src="https://jzrzw3o0xv.codesandbox.io/random.js"></script>
<script src="https://jzrzw3o0xv.codesandbox.io/engine3/math3.js"></script>
<script src="https://jzrzw3o0xv.codesandbox.io/engine3/engine3.js"></script>

Note: The random.js was not developed by me. It's just a seed generator.

Now, you can start doing things like creating visual objects. Use E.Obj([x, y], layer, color) to create a visual object. Note: color is an object (js object), not a string.

See the Pen qGxLdB by Trevor (@theangrybagel) on CodePen.

In the exampe above, we start by creating a new E.Obj. It's position is an array, [5, 5], representing [x, y]. Then, we use setTimeout to create a delay between each bit of code. The first thing that happens is the modification of o1's position. This property can be accessed with o1.pos. It returns an array, representing a position, which can be modified to change the objects position.

We then move the camera, which can be accessed with E.Camera.TargetPos. Note: E.Camera.Position should not be modified. The Camera's target position is also stored as an array, and upon modification, the Camera will automatically smooth to it's target position.

Next, we change the objects color through o1.color. This property is stored as an rgb object, which can be made with new rgb(r, g, b). Then, we use o1.moveTo() in order to smoothly move o1 to a desired position.
