Users


adduser username - prompts for a password and other info. Then adds a new user.

user files
/etc/passwd - doesn't store passwords, provides placeholder (x) for password. Shows info about users, their uids, etc. Other users located at the bottom of the file.
/etc/shadow - contains password hashes. You can crack hashes (unshadowing) with John or Hashcat.

su username - switch user
only sudoers can view /etc/shadow.
attempts to view shadow will be logged/reported to a file in /var/log/auth.log or /var/log/syslogs or something
