nmap -T4 ipaddress

add more depth
nmap -T4 -A -p- ipaddress
-A - all (os detection, version detection, script scanning and traceroute)
-p port
-p- all ports

-p80,443,512 - scans ports 80, 443, and 512
-oA filename - outputs all to filename.txt

ls /usr/share/nmap/scripts will list all the scripts for nmap

nmap -p 443 --script=ssl-enum-ciphers - note: setting script to all will run every script



nmap --script=ssl-enum-ciphers -p 443 192.168.202.130
gives you a bunch of ciphers for that IP
there's a least strength cipher at the bottom of the output
